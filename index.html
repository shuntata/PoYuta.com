コードの紹介の仕方

写真を使ってわかりやすくコードを理解できるようにしてます。 例えば、下の写真は文字の色を変えるコードです。 


  h1 {
   color: #000;
   }


文字の装飾

次は文字にいろいろな装飾を加える方法です。

色の変え方

色を変える方法は主に2つあり、1つ目はcssでcolorを使う方法です。 もう1つは、htmlのほうで<font>を使う方法もあります。


colorを使うやり方。今回のコードでは色コードで#71d219←この色にしています。


  h1 {
   color: #71d219;
  }

<font>を使う方法


  <h1>
   <font  color="#71d219">こんにちは<font>
  </h1>

色指定について

色の指定は#の後ろにつく6桁の色コードで指定できます。例：黒=#000　白=#fff

下のサイトは自分の使いたい色を細かく設定してそれを色コードに変換してくれる優れものです！！

https://itsakura.com/html-color-codes

文字の太さ

文字の太さはfont-weightを使います。太くしたいなら boldで細くしたいならlighterを使います。


  h1 {
   font-weight: bold;
  }

文字の大きさ

文字の大きさはfont-sizeで、数字pxで指定できます。

コードは次の通りです。今回は20pxでコードを書いてます。 


  h1 {
   font-size: 20px;
  }
